# Strategic Product Placement Analysis using Tableau and Flask

## Project Overview
This project analyzes the relationship between product placement, pricing strategies, promotions, consumer demographics, and sales performance. Using Tableau, we created interactive dashboards and story visualizations to uncover insights about how product positioning influences consumer behavior and sales volume.

The dashboard is published on Tableau Public and integrated into a simple web interface using Flask.

## Objectives
- Analyze how product placement affects sales performance
- Identify relationships between price, competitor price, and sales volume
- Understand the impact of promotions and seasonal products
- Study consumer demographics and foot traffic influence
- Build an interactive dashboard and story using Tableau
- Integrate the dashboard into a web interface using Flask
